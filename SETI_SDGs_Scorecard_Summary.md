# SETI for SDGs Scorecard - Executive Summary

## What is SETI?
**Science, Engineering, Technology and Innovation (SETI)** - A framework developed to assess how science and technology projects contribute to achieving the UN Sustainable Development Goals (SDGs).

## Philippine Leadership ðŸ‡µðŸ‡­
The **Philippines spearheaded this global initiative** through:
- **Department of Science and Technology Region XI (DOST XI)**
- **HELP Davao Network**
- Initial pilot in Davao Region (2016-2018)
- Expanded from 7 SDGs to all 17 SDGs after UNESCO Jakarta adoption (December 2018)

## Why DOST XI Developed This Tool

### Key Drivers:
1. **AP-FAST Project Opportunity** - Malaysian-funded UNESCO project in Davao Region
2. **Measurement Gap** - No existing tool to assess SETI contributions to SDGs
3. **Resource Optimization** - Need to prioritize limited funding for maximum impact
4. **Evidence-Based Planning** - Justify SETI investments with concrete results
5. **Regional Leadership** - Position Philippines as SDG implementation innovator

## Core Objectives

| Objective | Description |
|-----------|-------------|
| **Assessment Tool** | Systematically evaluate SETI projects' SDG contributions |
| **Gap Identification** | Identify where more SETI investment is needed |
| **Evidence-Based Decisions** | Quantify direct/indirect impacts on SDG achievement |
| **Cross-Sectoral Integration** | Address interconnected development challenges |
| **Accountability Framework** | Track progress transparently |

## Key Features

### Scoring System:
- **1-9 Responsiveness Scale**
  - 7-9: Highly responsive to SDGs
  - 4-6.99: Moderately responsive
  - 1-3.99: Slightly responsive

### Impact Classification:
- **Direct Impact** - Output directly contributes to SDG indicators
- **Indirect Impact** - Output indirectly supports SDG achievement

## Implementation Formula

The SETI Scorecard uses a specific formula to calculate project responsiveness:

```
RESPONSIVENESS SCORE = [(Likelihood Ã— Relevance) Ã— (1 if Direct Impact; 0.5 if Indirect Impact)]
```

Where:
- **Likelihood** = Progress towards completion (1-3 scale)
- **Relevance** = Contribution to SDG attainment (1-3 scale)
- **Impact Factor** = 1.0 for direct, 0.5 for indirect contribution
